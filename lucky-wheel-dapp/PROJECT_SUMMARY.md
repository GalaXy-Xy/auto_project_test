## 🎲 游戏机制

### 奖池分配
- **一等奖 (1名)**: 奖池的50%
- **二等奖 (2名)**: 奖池的20%
- **三等奖 (5名)**: 奖池的10%
- **服务费**: 奖池的20%

### 抽奖流程
1. 用户连接钱包
2. 支付0.01 ETH抽奖费用
3. 合约请求Chainlink VRF随机数
4. 根据随机数确定中奖结果
5. 自动分配奖金到中奖者账户

## 🛠️ 技术栈

### 区块链技术
- **智能合约**: Solidity 0.8.19
- **开发框架**: Hardhat
- **随机数**: Chainlink VRF
- **测试网**: Sepolia

### 前端技术
- **框架**: Next.js 14
- **UI库**: React 18 + Tailwind CSS
- **动画**: Framer Motion
- **区块链交互**: wagmi + ethers.js

### 开发工具
- **版本控制**: Git + GitHub
- **自动化**: 自定义脚本
- **测试**: Chai + Mocha
- **部署**: 多网络支持
